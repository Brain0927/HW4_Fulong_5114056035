# AI Chef Assistant - 專案摘要報告

## ABSTRACT

本專案開發了一款基於 Google Gemini API 的 AI 廚師助手 Streamlit 應用程式，具有兩大核心功能模組。

**應用功能**：第一模組為「AI 廚師對話」，使用者可針對烹飪技巧、食材搭配、營養資訊等進行即時互動查詢，AI 會根據用戶提問給予專業建議。第二模組為「食譜生成器」，使用者輸入菜名、難度級別、份數、烹飪時間及食材清單，系統自動生成完整食譜，包括步驟說明、營養分析等結構化資訊。

**技術架構**：應用採用 Streamlit 作為 Web UI 框架，整合 Google Gemini 2.5-Flash 模型進行自然語言處理。核心模組 `ai_chef_functions.py` 封裝了 AI 交互邏輯，包含自動服務檢測機制、多模型備援機制及完整錯誤處理。

**部署架構**：專案透過 Git 版本控制系統部署至 GitHub 儲存庫，設定 `.gitignore` 保護 API 金鑰安全，環境變數透過 `.env` 檔案配置。應用包含多項診斷工具用於 API 連接驗證及故障排除。

**使用者體驗**：介面採用雙語設計（中英混合），提供即時對話歷史、一鍵清空功能、快速示例按鈕及完整的錯誤提示。CSS 自訂樣式提供視覺區隔，Tab 標籤具備漸層背景、懸停效果等互動視覺回饋。

**產出成果**：完整的生產級應用程式、詳細技術文檔、API 診斷工具組及 GitHub 部署管道。本專案展示了現代 AI 應用開發的最佳實踐，包括模組化設計、環境隔離、版本控制及使用者介面最佳化。
